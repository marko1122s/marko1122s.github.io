# Marko Trajkovski — Portfolio

Persönliche Portfolio-Website (Bewerbungen) von Marko Trajkovski — Web Developer & IT-/Business-Absolvent aus Wien.

Gebaut mit **HTML5, CSS3 und Vanilla JavaScript** — ohne Frameworks.

## Features
- Dark Mode (Standard) + Light-Toggle
- Zweisprachig: Deutsch / English (Umschalter)
- Scroll-Reveal-Animationen, smooth scrolling, Hover-Effekte
- 5 Projekte mit interaktiven Mini-Demos (Dashboard, Finanztabelle, Wetter, ERD, Desktop-App)
- Projekt-Detail-Modals
- Funktionierendes Kontaktformular (Web3Forms)
- Voll responsive (Mobile First)

## Struktur
```
index.html          # Seite
css/style.css       # Styles
js/script.js        # Logik (Theme, i18n, Demos, Modal, Formular)
assets/             # CV (PDF), Profilbild, Favicon
```

## Lokal starten
Einfach `index.html` öffnen — oder ein lokaler Server:
```bash
python3 -m http.server 8000
```
Dann http://localhost:8000 aufrufen.
